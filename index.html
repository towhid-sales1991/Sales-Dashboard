Setup & Connection

Connect this dashboard to a shared Google Sheet so all colleagues see the same live data.

1
Create a Google Sheet with a Sales tab (headers: PI Date, Proforma Invoice No, Customer Name, Buyer Name, PI Value, Factory, Sales Person Name, New Sales) and a Targets tab (headers: Sales Person, Year, Month, Target Value (USD)).
2
Deploy an Apps Script Web App bound to that sheet, and paste its /exec URL below.
